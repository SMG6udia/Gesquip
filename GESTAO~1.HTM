Configuração do Firebase

Cole aqui o JSON de configuração do seu projeto Firebase. Inclua o campo databaseURL.

Guia de configuração passo a passo
Passo 1 — Criar projeto no Firebase (grátis)
  1. Acesse console.firebase.google.com
  2. Clique em "Criar um projeto"
  3. Dê um nome (ex: gestao-equipamentos) e conclua o assistente
Passo 2 — Ativar o Realtime Database
  1. No menu lateral esquerdo, clique em Build → Realtime Database
  2. Clique em "Criar banco de dados"
  3. Escolha a região mais próxima (ex: us-central1)
  4. Inicie no modo de teste (permite leitura/escrita por 30 dias — depois ajuste as regras)
Passo 3 — Ajustar as regras de segurança

No Realtime Database → aba Regras, cole e publique:

{ "rules": { ".read": true, ".write": true } }

⚠️ Mantenha a URL do banco privada. Apenas quem tiver o link do sistema poderá acessar.

Passo 4 — Obter o JSON de configuração
  1. No painel do projeto, clique na engrenagem ⚙️ → Configurações do projeto
  2. Role até "Seus aplicativos" → clique em "Adicionar app" → Web (</>)
  3. Registre o app (qualquer nome) — o Firebase vai mostrar um objeto firebaseConfig
  4. Copie apenas o JSON interno (entre { })
  5. Adicione manualmente o campo "databaseURL" — você acha a URL na página do Realtime Database
Passo 5 — Publicar no GitHub Pages (link de acesso)
  1. Crie um repositório no GitHub (pode ser privado ou público)
  2. Faça upload do arquivo gestao-equipamentos.html
  3. Renomeie-o para index.html no repositório
  4. Vá em Settings → Pages → Branch: main → / (root) → Save
  5. Em alguns minutos seu sistema estará em: https://seu-usuario.github.io/nome-do-repo/
  6. Configure o Firebase em cada dispositivo colando o JSON uma única vez — o sistema salva a configuração no navegador
Dica: Depois de configurar o Firebase em um computador, todos que acessarem o link do GitHub Pages precisarão colar o JSON de configuração apenas na primeira vez. Após isso, o sistema lembra automaticamente.
