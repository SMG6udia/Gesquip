Como configurar (feito 1 vez por toda a equipe):
1. Abra o arquivo apps-script.gs e siga as instruções dentro dele
2. Ao implantar, defina Quem tem acesso: Qualquer pessoa (não "na organização")
3. Copie a URL do Aplicativo da Web gerada
4. Cole abaixo e clique em Salvar e Testar
5. Compartilhe o link do GitHub Pages com todos — dados sincronizados a cada 15s ✅
⚠️ Acesso configurado como "na organização"?
O GitHub Pages não consegue autenticar com o Google automaticamente. Reimplante o Apps Script com Quem tem acesso: Qualquer pessoa.

Limite = máx de aparelhos desse tipo por colaborador. 0 = sem limite.

Atenção: estas ações não podem ser desfeitas.
